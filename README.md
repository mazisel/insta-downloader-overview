# Insta Downloader

Instagram içeriklerini düzenli ve profesyonel şekilde arşivleyen Python aracı.

## Tech Stack
- Python
- Instaloader
- Requests

## Özellikler
- Toplu fotoğraf, video ve carousel indirme
- Gönderi açıklamalarını ayrı dosyada saklama
- Otomatik ve düzenli klasör yapısı
- Rate limiting ile güvenli indirme

Detaylı bilgi ve işbirliği için iletişime geçin.
