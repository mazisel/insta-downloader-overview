# Insta Downloader

**🇹🇷 Türkçe**

Instagram içeriklerini düzenli ve profesyonel şekilde arşivleyen Python aracı.

## Tech Stack
- Python
- Instaloader
- Requests

## Özellikler
- Toplu fotoğraf, video ve carousel indirme
- Gönderi açıklamalarını ayrı dosyada saklama
- Otomatik ve düzenli klasör yapısı
- Rate limiting ile güvenli indirme

---

**🇬🇧 English**

A Python tool that archives Instagram content in an organized and professional manner.

## Tech Stack
- Python
- Instaloader
- Requests

## Features
- Bulk photo, video and carousel downloading
- Storing post descriptions in separate files
- Automatic and organized folder structure
- Secure downloading with rate limiting
